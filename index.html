Flood Monitoring System Project

AIM

To build a prototype of an IoT system using Thingspeak cloud and Raspberry Pi and develop a Flood Monitoring System. The aim of the Flood Monitoring System project is to develop a real-time monitoring system using Raspberry Pi and various sensors to detect and alert against potential flooding conditions.

Hardware Requirements

  • Raspberry Pi
  • Ultrasonic Sensor
  • Temperature Sensor
  • Rain Drop Sensor
  • Buzzer
  • LEDs

Software Requirements

  • Raspberry Pi
  • ThingSpeak Cloud
  • Android Studio
  • Python v3.7

Theory

A flood monitoring system is designed to detect and monitor water levels, rainfall, and other environmental factors in areas prone to flooding. The system provides early warning alerts, aids in disaster management, and mitigates the impact of floods.

ThingSpeak Overview

ThingSpeak is an open-source IoT platform that stores and retrieves data from devices using HTTP or MQTT protocols. It provides data visualization and analysis tools with integrated MATLAB support.

Algorithm

  1. Power up the Raspberry Pi module and make connections as per the layout diagram.
  2. Program the Raspberry Pi to sense distance, rain status, and humidity at regular intervals.
  3. Create a ThingSpeak channel and generate an API key.
  4. Upload sensed data to ThingSpeak, which is visualized in real-time.
  5. Develop a Flutter-based web/app to visualize the project overview and sensor data.

Conclusion

A prototype of an IoT-based Flood Monitoring System using ThingSpeak and Raspberry Pi has been successfully developed, and real-time readings have been visualized through the system.
